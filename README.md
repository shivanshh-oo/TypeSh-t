## ✦ How It Works
 
### ASCII Conversion Pipeline
 
```
Source (video/image)
       │
       ▼
  Canvas (hidden)            ← Scaled to `resolution` columns
       │                        Height = cols / srcAspect × 0.5
       ▼                        (0.5 = monospace char aspect ratio)
  getImageData()             ← Raw RGBA pixel array
       │
       ▼
  Luminance per pixel        ← ITU-R BT.601: 0.299R + 0.587G + 0.114B
       │
       ▼
  brightnessToChar()         ← Maps 0–255 → index in charset string
       │
       ▼
  asciiOutput (pre element)  ← Styled with chosen color/bg/fontSize
```
 
### Character Aspect Correction
 
Monospace characters are approximately **0.5× as wide as they are tall**. Without correction, the ASCII output would appear vertically stretched. The renderer compensates by calculating:
 
```
rows = Math.round(cols / sourceAspectRatio × CHAR_ASPECT)
```
 
where `CHAR_ASPECT = 0.5`.
 
### Live Camera Loop
 
```javascript
function frame() {
  if (!videoStream) return;
  applyAscii(imageToAscii(video)); // draw video frame → canvas → ASCII
  animFrameId = requestAnimationFrame(frame);
}
```
 
The loop is cancelled on camera stop or tab switch to avoid unnecessary GPU/CPU usage.

## ✦ Character Sets
 
| Name | Characters | Best For |
|---|---|---|
| `standard` | 70-char dense ramp | Photos, portraits |
| `blocks` | `█▓▒░ ` | Bold, high-contrast art |
| `simple` | ` .:-=+*#@` | Clean, minimal look |
| `letters` | A–Z, a–z, 0–9 | Text-art feel |
| `binary` | `10 ` | Hacker/matrix aesthetic |
| `custom` | User-defined | Full creative control |
 
For custom sets, enter characters from **darkest to lightest**. The renderer maps luminance 0 (black) → first character, 255 (white) → last character.

## ✦ Export Details
 
### PNG Export
 
The PNG exporter creates an off-screen `<canvas>` that:
1. Fills the background with your chosen color
2. Renders each ASCII line using `fillText()` at `"Courier New"` monospace
3. Sizes the canvas precisely: `charWidth × columns + padding`
4. Triggers a download via a temporary `<a>` element
Files are named `ascii-art-{timestamp}.png`.
 
### Text Copy
 
Uses the modern `navigator.clipboard.writeText()` API with a `document.execCommand('copy')` fallback for older browsers.

## ✦ Customisation Guide
 
### Change Default Character Set
 
In `app.js`, edit the `CHARSETS` object:
 
```javascript
const CHARSETS = {
  standard: `$@B%8&WM#...`,  // ← edit or add your own
  mySet: '░▒▓█ ',
};
```
 
Then add the option to the `<select>` in `index.html`:
 
```html
<option value="mySet">My Custom Set</option>
```
